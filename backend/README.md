## Setup
To run this app on your local machine install dependencies using pip:
```
pip install -r requirements.txt
```
Create superuser:
```
python manage.py createsuperuser
```
Make migrations:
```
python manage.py migrate
```
Run:
```
python manage.py runserver
```
